# Introduction

The Albion API powers the [Albion platform](https://platform.albion.rest/), providing programmatic access to swap, trade, and manage tokenized assets on the Base network. Through the API you can:

- Discover available tokens
- Get swap quotes and generate transaction calldata
- Deploy and manage DCA orders
- Monitor trades

**Base URL:** `https://api.albion.rest`

**Full API Reference:** The interactive Swagger UI is available at [https://api.albion.rest/swagger/](https://api.albion.rest/swagger/) with complete request/response schemas for every endpoint.

This guide focuses on the typical workflows and how the endpoints connect.
